# Restaurant Menu and Order Management System — Part 2

## Overview

This project is a command-line Restaurant Order Management System built using core Python data structures such as dictionaries, lists, and nested combinations. It simulates menu exploration, cart operations, inventory tracking, and sales analysis.

## Concepts Used

Dictionaries and nested dictionaries
Lists and list operations
Loops (for, while)
Conditional statements
Functions
Deep copy using copy module
Basic data analysis

## Tasks Implemented

Task 1 — Explore the Menu
Displayed menu grouped by category
Calculated total items and available items
Identified most expensive item
Listed items priced under 150

Task 2 — Cart Operations
Implemented add, remove, and update logic
Handled unavailable and invalid items
Avoided duplicate entries by updating quantity
Generated final order summary with subtotal, GST, and total

Task 3 — Inventory Tracker
Created deep copy of inventory
Demonstrated that backup remains unchanged after modification
Simulated stock deduction based on cart
Generated reorder alerts for low stock items

Task 4 — Sales Log Analysis
Calculated total revenue per day
Identified best-selling day
Found most ordered item
Updated sales log with new data and recalculated metrics
Printed numbered list of all orders
